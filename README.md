# maze-game
Есть поле размером 3 на 3 ячейки. В начале игры в случайную ячейку помещаем маркер. Далее генерируются 10 «ходов» (возможные варианты «вверх», «влево», «вниз», «вправо»). Игрок должен в уме «пройти» по этим ходам по лабиринту и указать конечную точку маркера.
После ответа (клик на ячейку) идет проверка ответа.
